# RULES & GUIDELINES

## Non-negotiables
1) Preserve functionality and screens.
2) No surprise dependencies: document any new one in `docs/DEPENDENCIES.md`.
3) WebView debugging ONLY in debug builds.
4) Keep web UI under `assets/www/`.
5) Mobile-first, always.

## If something is unclear
Add a TODO + a task in `docs/TASK_BOARD.md`. Do not improvise a redesign.
